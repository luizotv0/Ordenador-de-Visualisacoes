# 🎵 Ordenador de Visualizações
## 📄 Descrição
Esse é um programa em C desenvolvido para registrar e organizar o número de visualizações de músicas. Ele utiliza vetores e aplica o método Bubble Sort para ordenar os dados em ordem crescente e decrescente, permitindo identificar quais músicas têm mais ou menos visualizações.

Ideal para fins educativos, o projeto demonstra conceitos fundamentais de programação, como manipulação de vetores, laços de repetição, estruturas condicionais e funções.

## ⚙️ Funcionalidades
🎶 Registro das visualizações de 10 músicas (até a casa de bilhões)
📊 Exibição organizada do vetor original de visualizações
🔼 Ordenação em ordem crescente
🔽 Ordenação em ordem decrescente
🖥️ Interface interativa via terminal

## 🛠️ Estruturas utilizadas

⚖️ Estruturas condicionais (if) para definir a ordem de troca e validar entradas  
🗂️ Vetor (long long) para armazenar as visualizações  
🔁 Função bubbleSort para ordenação crescente e decrescente  
🖨️ Função imprimir para exibir os dados  

## ⚠️ Observações
O programa não persiste os dados: todas as informações são armazenadas apenas durante a execução.
